# orm-pack

Mutualize ORM composer dependencies.

# Installation

```bash
composer require johnkrovitch/orm-pack
```

# Dependencies

```json
{
    "require": {
        "gedmo/doctrine-extensions": "*",
        "johnkrovitch/repository": "*",
        "stof/doctrine-extensions-bundle": "*",
        "symfony/orm-pack": "*",
        "babdev/pagerfanta-bundle": "*"
    }
}
```
