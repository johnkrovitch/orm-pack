# orm-pack

Mutualize ORM composer dependencies.

# Installation

```bash
composer require johnkrovitch/orm-pack
```

# Dependencies

```json
{
    "require": {
        "johnkrovitch/repository": "^1.0",
        "stof/doctrine-extensions-bundle": "^1.3",
        "symfony/orm-pack": "^1.0",
        "white-october/pagerfanta-bundle": "^1.3"
    }
}
```
